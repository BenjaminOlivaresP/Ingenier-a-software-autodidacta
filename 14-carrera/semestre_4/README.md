# Semestre 4 — Sistemas y Desarrollo Full Stack

Este semestre se centra en la integración de frontend y backend
para construir aplicaciones **full stack completas**.

Durante estas **16 semanas** se desarrollarán habilidades
para manejar datos, servicios, autenticación y despliegue,
creando aplicaciones web modernas y funcionales.

Tecnologías utilizadas:

- Node.js avanzado  
- Bases de datos relacionales y no relacionales  
- Express y frameworks backend  
- Frontend dinámico integrado (React o Vanilla JS avanzado)  
- Autenticación JWT y seguridad web  
- DevOps básico (despliegue local y en la nube)  

El objetivo del semestre es pasar de construir aplicaciones
backend aisladas a crear sistemas completos que gestionen
datos, servicios y funcionalidades tanto en el servidor
como en el cliente.

# Proyecto final del semestre

Aplicación:

Sistema full stack completo con frontend y backend integrados.

Funcionalidades posibles:

- gestión completa de usuarios y datos  
- autenticación y roles de acceso  
- consumo de APIs externas e internas  
- operaciones CRUD avanzadas  
- integración de frontend dinámico y backend robusto  
- despliegue local o en la nube  

El proyecto integrará:

- Node.js y Express  
- Base de datos relacional o no relacional  
- Frontend dinámico (React, Vue o Vanilla JS avanzado)  
- Autenticación JWT y seguridad básica  
- Buenas prácticas full stack y despliegue básico

# Resultado del semestre

Al finalizar el semestre el estudiante podrá:

- construir aplicaciones **full stack completas**  
- integrar frontend dinámico con backend robusto  
- diseñar y consumir APIs internas y externas  
- implementar autenticación y control de acceso  
- optimizar rendimiento y seguridad de aplicaciones  
- desplegar sistemas en entornos locales o remotos  

Esto prepara al estudiante para avanzar al **Año 3 — Ingeniería avanzada**,
donde se profundizará en arquitectura de software, escalabilidad
y despliegue profesional de aplicaciones.

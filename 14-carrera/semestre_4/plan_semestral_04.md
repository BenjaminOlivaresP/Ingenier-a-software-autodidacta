# Semestre 4 — Sistemas y Desarrollo Full Stack

Duración: 16 semanas  
Horas semanales: 30  
Total aproximado: 480 horas

---

# Objetivo del semestre

Aprender a integrar frontend y backend para construir aplicaciones
full stack completas y funcionales.

Durante este semestre el estudiante desarrollará habilidades en:

- desarrollo avanzado de servidores y APIs con Node.js  
- integración de frontend dinámico con backend  
- construcción de endpoints RESTful avanzados  
- autenticación y seguridad web con JWT  
- testing, optimización y despliegue de aplicaciones  
- buenas prácticas de organización de proyectos full stack

El objetivo es transformar los conocimientos de backend en habilidades
para construir sistemas completos, seguros y escalables.

Al finalizar el semestre el estudiante será capaz de:

- integrar frontend y backend en aplicaciones full stack  
- diseñar y consumir APIs internas y externas  
- implementar autenticación avanzada y control de acceso  
- optimizar rendimiento y seguridad de aplicaciones  
- probar, desplegar y mantener sistemas completos  
- construir proyectos listos para producción o portafolio

---

# Asignaturas del semestre

## Programación Full Stack Avanzada

Profundización en integración y desarrollo de sistemas completos.

Temas principales:

- Node.js avanzado y Express  
- Integración de frontend dinámico (React, Vue o Vanilla JS avanzado)  
- Rutas, middleware y control de flujo  
- POO y modularización en aplicaciones full stack  

Objetivo:

Desarrollar aplicaciones integradas y mantenibles
con frontend y backend coordinados.

---

## Estructuras de Datos y Algoritmos Intermedios

Optimización de la lógica y eficiencia de aplicaciones.

Temas principales:

- estructuras de datos comunes (listas, colas, pilas)  
- algoritmos de búsqueda y ordenamiento  
- eficiencia en operaciones full stack  
- patrones de diseño aplicables a aplicaciones completas  

Objetivo:

Mejorar la calidad y rendimiento de las aplicaciones
con lógica y estructuras optimizadas.

---

## Bases de Datos Avanzadas

Gestión de datos complejos y optimización de consultas.

Temas principales:

- bases de datos relacionales y no relacionales  
- diseño de relaciones avanzadas  
- consultas optimizadas y transacciones  
- integración con ORM y frameworks backend  

Objetivo:

Manejar información compleja de forma eficiente,
asegurando consistencia y rendimiento.

---

## Autenticación y Seguridad Web

Protección de sistemas y usuarios en aplicaciones full stack.

Temas principales:

- autenticación JWT  
- manejo de sesiones y roles  
- encriptación y hashing de contraseñas  
- protección de endpoints y middleware de seguridad  

Objetivo:

Garantizar la seguridad y control de acceso
en aplicaciones web modernas.

---

## Testing, Optimización y Despliegue

Aseguramiento de calidad y preparación para producción.

Temas principales:

- testing de frontend y backend integrados  
- optimización de rendimiento y tiempos de respuesta  
- despliegue en entornos locales y en la nube  
- control de versiones avanzado y CI básico  

Objetivo:

Preparar aplicaciones full stack para producción,
asegurando estabilidad y rendimiento.

---

## Organización de Proyectos Full Stack

Buenas prácticas para mantener proyectos complejos ordenados.

Temas principales:

- estructura de carpetas y modularización  
- separación de responsabilidades frontend/backend  
- documentación completa y profesional  
- escalabilidad y mantenibilidad de proyectos  

Objetivo:

Mantener proyectos integrados claros y escalables,
listos para colaboración o portafolio.

---

# Proyecto del semestre

Construcción de un sistema full stack completo.

Proyecto:

Aplicación web completa con frontend dinámico y backend robusto.

Funcionalidades posibles:

- endpoints CRUD avanzados  
- integración de frontend y backend  
- autenticación con roles y JWT  
- consumo de APIs internas y externas  
- despliegue local o en la nube  

Objetivos del proyecto:

- aplicar Node.js y frameworks frontend  
- manejar bases de datos complejas y relaciones avanzadas  
- implementar buenas prácticas full stack  
- construir un proyecto completo para portafolio o producción

El desarrollo del proyecto se realizará
durante las últimas semanas del semestre
para integrar todos los conocimientos aprendidos.

---

# Resultados esperados al finalizar el semestre

Al finalizar el semestre el estudiante tendrá:

- dominio de Node.js avanzado y desarrollo full stack  
- capacidad para integrar frontend dinámico con backend robusto  
- comprensión avanzada de estructuras de datos y algoritmos  
- habilidad para implementar autenticación y seguridad web  
- experiencia en testing, optimización y despliegue de aplicaciones  
- proyectos full stack completos listos para producción o portafolio

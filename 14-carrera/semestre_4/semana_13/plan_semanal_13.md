# Semana 13 — Integración continua y control de versiones avanzado

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a mantener proyectos full stack organizados mediante control de versiones avanzado y pipelines de integración continua (CI).  
El estudiante desarrollará habilidades para automatizar pruebas, garantizar consistencia en el código y facilitar despliegues confiables.

Durante la semana se estudiará:

- control de versiones avanzado con Git y GitHub  
- ramas, merge, rebase y resolución de conflictos  
- pipelines de integración continua (CI)  
- automatización de tests y despliegues  
- buenas prácticas de flujo de trabajo colaborativo  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- manejar repositorios complejos con Git y GitHub  
- organizar ramas y gestionar merges correctamente  
- configurar pipelines de CI para pruebas y despliegue  
- automatizar tests en cada push o pull request  
- mantener consistencia y calidad en proyectos colaborativos  
- documentar y versionar aplicaciones de manera profesional

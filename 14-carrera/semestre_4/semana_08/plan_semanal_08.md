# Semana 8 — Consumo de APIs internas y externas

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a consumir y manejar **APIs internas y externas** dentro de aplicaciones full stack.  
El estudiante desarrollará habilidades para integrar servicios, obtener y enviar datos, y conectar el frontend con el backend de manera eficiente.

Durante la semana se estudiará:

- conceptos de APIs y servicios web  
- métodos HTTP y status codes  
- consumo de APIs internas (propias)  
- consumo de APIs externas (terceros)  
- manejo de errores y validación de datos  
- integración con frontend dinámico  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- consumir APIs internas y externas desde Node.js y frontend  
- realizar peticiones GET, POST, PUT y DELETE correctamente  
- manejar respuestas y errores de forma segura  
- validar y transformar datos recibidos de APIs  
- integrar datos de APIs externas en aplicaciones full stack  
- documentar la integración y flujo de datos

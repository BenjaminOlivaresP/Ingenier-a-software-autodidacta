# Semana 2 — Endpoints CRUD avanzados y conexión con frontend

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Profundizar en la creación de **endpoints CRUD avanzados** y su integración con frontend dinámico.  
El estudiante aprenderá a manipular datos complejos y garantizar consistencia y seguridad en la comunicación entre cliente y servidor.

Durante la semana se estudiará:

- operaciones CRUD avanzadas (CREATE, READ, UPDATE, DELETE)  
- consultas complejas a bases de datos relacionales  
- validación de datos y manejo de errores  
- integración de frontend con endpoints mediante fetch/AJAX  
- pruebas de endpoints con Postman y navegador  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- implementar endpoints CRUD avanzados con Node.js y Express  
- validar y sanitizar datos de entrada  
- integrar frontend dinámico con backend  
- probar y depurar endpoints de manera profesional  
- documentar rutas y flujo de datos de manera clara

# Semana 7 — ORM y manejo de datos con frameworks

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a usar **ORMs (Object-Relational Mapping)** y frameworks de datos para manejar bases de datos de manera más eficiente en aplicaciones full stack.  
El estudiante desarrollará habilidades para abstraer consultas SQL, mantener consistencia de datos y optimizar la interacción entre backend y bases de datos.

Durante la semana se estudiará:

- conceptos de ORM y ventajas frente a SQL directo  
- frameworks populares: Sequelize, TypeORM, Mongoose  
- mapeo de tablas y modelos de datos  
- consultas avanzadas con ORM  
- relaciones entre modelos y transacciones  
- integración con aplicaciones Node.js y frontend  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- crear modelos de datos usando ORM  
- definir relaciones entre modelos (1:1, 1:N, N:N)  
- realizar operaciones CRUD con abstracción de SQL  
- manejar transacciones y validaciones desde el ORM  
- integrar modelos con rutas y controladores de Node.js  
- usar un ORM como capa de datos en proyectos full stack

# Semana 14 — Proyecto intermedio: Aplicación full stack funcional

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aplicar todos los conocimientos adquiridos hasta la fecha para desarrollar un proyecto full stack intermedio.  
El estudiante integrará frontend dinámico, backend robusto, base de datos y control de versiones, consolidando habilidades prácticas en un entorno realista.

Durante la semana se trabajará en:

- planificación y diseño de la aplicación  
- integración de frontend y backend  
- consumo de APIs internas y externas  
- manejo de bases de datos y endpoints CRUD  
- pruebas, control de versiones y pipeline de CI  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- diseñar y estructurar un proyecto full stack funcional  
- integrar frontend dinámico con backend y base de datos  
- implementar endpoints RESTful y CRUD completos  
- manejar autenticación básica y roles de usuario  
- aplicar testing, control de versiones y CI  
- documentar el proyecto para uso y mantenimiento

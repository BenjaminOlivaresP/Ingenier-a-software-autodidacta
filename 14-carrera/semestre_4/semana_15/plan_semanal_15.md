# Semana 15 — Desarrollo del proyecto final

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Consolidar todos los conocimientos adquiridos en el semestre desarrollando un **proyecto full stack final completo**.  
El estudiante aplicará integración frontend-backend, autenticación, bases de datos y buenas prácticas de desarrollo, preparando un sistema funcional listo para producción o portafolio.

Durante la semana se trabajará en:

- planificación detallada y organización de tareas finales  
- desarrollo de funcionalidades avanzadas  
- integración de frontend y backend con base de datos  
- implementación de autenticación y roles  
- pruebas, documentación y control de versiones  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- desarrollar un proyecto full stack completo y funcional  
- implementar autenticación y control de accesos  
- manejar datos complejos y relaciones entre entidades  
- integrar frontend dinámico con backend seguro  
- probar y depurar la aplicación de manera profesional  
- documentar todo el proyecto para uso y mantenimiento

# Evaluación personal

Responder:

1. ¿Mi proyecto final integra correctamente frontend, backend y base de datos?  
2. ¿Los endpoints y funcionalidades avanzadas funcionan correctamente y de manera segura?  
3. ¿La UI es dinámica, responsiva y comunica correctamente con el backend?  
4. ¿El proyecto está documentado, versionado y listo para entrega profesional?  

Si la mayoría es **sí**, el estudiante está listo para continuar con la **Semana 16 — Finalización y presentación del proyecto**.

# Malla Curricular — Ingeniería de Software Autodidacta

Autor: Benjamín Olivares  

Duración estimada: 3 años  
Total: 6 semestres  

Este documento define la estructura completa del programa de estudio
de ingeniería de software autodidacta.

El objetivo de esta carrera es desarrollar las habilidades necesarias
para diseñar, construir y mantener sistemas de software modernos,
abarcando frontend, backend, bases de datos, arquitectura de software
y despliegue de aplicaciones.

---

# Estructura de la carrera

La carrera está dividida en tres años de estudio.

Cada año profundiza en un nivel distinto de dominio del software:

Año 1 — Fundamentos  
Año 2 — Desarrollo profesional  
Año 3 — Ingeniería avanzada

Cada semestre incluye:

- fundamentos teóricos
- ejercicios técnicos
- micro-proyectos
- un proyecto integrador

El aprendizaje sigue una metodología basada en:

- construcción de proyectos
- resolución de problemas
- documentación del aprendizaje
- desarrollo progresivo de habilidades de ingeniería

---

# Año 1 — Fundamentos del Software

El objetivo del primer año es construir una base sólida en
pensamiento computacional, lógica de programación y desarrollo web.

---

## Semestre 1 — Fundamentos de Ingeniería de Software

Objetivo:

Comprender cómo funciona la computación moderna y desarrollar
las primeras habilidades de programación y desarrollo web.

Asignaturas principales:

- Historia de la computación
- Pensamiento computacional
- Lógica matemática para programación
- Fundamentos de programación
- Introducción al desarrollo web
- Herramientas del programador

Tecnologías:

HTML  
CSS  
JavaScript básico  

Herramientas:

Visual Studio Code  
Git  
GitHub  

Proyecto del semestre:

Aplicación web simple de seguimiento de hábitos.

---

## Semestre 2 — Programación y desarrollo web

Objetivo:

Profundizar en la programación y comenzar a construir aplicaciones
web más complejas.

Asignaturas principales:

- Programación estructurada
- JavaScript avanzado
- HTML y CSS avanzado
- Introducción a algoritmos
- Introducción a bases de datos
- Arquitectura básica de aplicaciones web

Tecnologías:

JavaScript  
Node.js (introducción)  
SQL (fundamentos)

Proyecto del semestre:

Aplicación web completa con interacción dinámica.

---

# Año 2 — Desarrollo profesional

El segundo año se enfoca en construir aplicaciones completas
y comprender cómo funcionan los sistemas web modernos.

---

## Semestre 3 — Backend y arquitectura web

Objetivo:

Aprender a construir sistemas backend y comprender
la arquitectura de aplicaciones web modernas.

Asignaturas principales:

- Programación orientada a objetos
- Backend con Node.js
- Bases de datos SQL
- APIs y arquitectura web
- Control de versiones avanzado

Tecnologías:

Node.js  
Express  
SQL  

Proyecto del semestre:

Sistema backend completo con API.

---

## Semestre 4 — Sistemas y desarrollo full stack

Objetivo:

Integrar frontend y backend para construir aplicaciones
full stack completas.

Asignaturas principales:

- Estructuras de datos
- Algoritmos intermedios
- Backend avanzado
- Autenticación y seguridad web
- Introducción a DevOps

Tecnologías:

Node.js avanzado  
Bases de datos  
Autenticación JWT  

Proyecto del semestre:

Aplicación full stack completa.

---

# Año 3 — Ingeniería avanzada

El tercer año se enfoca en arquitectura de software,
escalabilidad de sistemas y desarrollo profesional.

---

## Semestre 5 — Arquitectura de software

Objetivo:

Aprender a diseñar sistemas complejos
y aplicar principios de ingeniería de software profesional.

Asignaturas principales:

- Arquitectura de software
- Patrones de diseño
- Microservicios
- Sistemas distribuidos
- Testing profesional
- Optimización de software

Tecnologías:

Docker  
Testing frameworks  
Arquitectura de servicios

Proyecto del semestre:

Sistema de software profesional.

---

## Semestre 6 — Ingeniería de sistemas escalables

Objetivo:

Desarrollar la capacidad de diseñar y desplegar
sistemas de software robustos y escalables.

Asignaturas principales:

- Cloud computing
- Escalabilidad de sistemas
- Seguridad avanzada
- Inteligencia artificial aplicada
- Despliegue de aplicaciones

Tecnologías:

Cloud platforms  
Infraestructura moderna  

Proyecto final:

Construcción de una aplicación profesional completa
lista para producción y portafolio.

---

# Objetivo final de la carrera

Al finalizar este programa autodidacta el estudiante será capaz de:

- diseñar aplicaciones web completas
- desarrollar sistemas backend y frontend
- trabajar con bases de datos modernas
- comprender arquitectura de software
- construir productos digitales funcionales
- desplegar aplicaciones en entornos reales

# Malla Curricular — Ingeniería de Software Autodidacta

Duración: 3 años  
Total: 6 semestres  

Objetivo: dominar desarrollo de software moderno, backend, frontend,
sistemas, algoritmos y arquitectura de software.

---

# Año 1 — Fundamentos

## Semestre 1
- Historia de la computación
- Pensamiento computacional
- Lógica matemática para programación
- Fundamentos de programación
- Introducción al desarrollo web
- Herramientas del programador (Git, terminal, VSCode)

## Semestre 2
- Programación estructurada
- JavaScript profundo
- HTML y CSS avanzado
- Introducción a algoritmos
- Introducción a bases de datos
- Proyecto web completo

---

# Año 2 — Desarrollo profesional

## Semestre 3
- Programación orientada a objetos
- Backend con Node.js
- Bases de datos SQL
- APIs y arquitectura web
- Git avanzado
- Proyecto backend completo

## Semestre 4
- Estructuras de datos
- Algoritmos intermedios
- Backend avanzado
- Autenticación y seguridad web
- Introducción a DevOps
- Proyecto full stack

---

# Año 3 — Ingeniería avanzada

## Semestre 5
- Arquitectura de software
- Microservicios
- Sistemas distribuidos
- Testing profesional
- Optimización de software
- Proyecto profesional

## Semestre 6
- Cloud computing
- Escalabilidad de sistemas
- Seguridad avanzada
- Inteligencia artificial aplicada
- Proyecto final

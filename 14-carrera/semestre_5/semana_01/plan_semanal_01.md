# Semana 1 — Introducción a la Arquitectura de Software

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Comprender qué es la **arquitectura de software** y por qué es una disciplina fundamental en la ingeniería moderna.

El estudiante aprenderá a diferenciar entre escribir código y **diseñar sistemas completos**, entendiendo cómo se organizan los componentes de un sistema para garantizar escalabilidad, mantenibilidad y rendimiento.

Durante la semana se trabajará en:

- concepto de arquitectura de software  
- responsabilidades de un arquitecto de software  
- estructura de sistemas modernos  
- componentes principales de una aplicación  
- documentación arquitectónica

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- comprender qué es la arquitectura de software  
- identificar componentes estructurales de un sistema  
- analizar cómo se organizan aplicaciones modernas  
- documentar la arquitectura de un sistema simple  
- diferenciar entre código y diseño estructural

# Semana 12 — Manejo de Datos en Aplicaciones

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a gestionar la información dentro de aplicaciones web, combinando datos de APIs, arrays, objetos y almacenamiento temporal para construir interfaces más complejas y dinámicas.

Durante la semana se estudiará:

- manipulación avanzada de arrays y objetos  
- filtrado, búsqueda y ordenamiento de datos  
- sincronización de datos con el DOM  
- almacenamiento temporal con variables y estructuras  
- preparación para persistencia con LocalStorage  

---

# Habilidades que se desarrollarán

Al finalizar la semana el estudiante debería poder:

- procesar datos recibidos de APIs o formularios  
- filtrar, ordenar y transformar información  
- actualizar interfaces con datos dinámicos  
- preparar datos para almacenamiento persistente  
- documentar la manipulación y flujo de datos

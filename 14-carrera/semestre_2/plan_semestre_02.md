# Semestre 2 — JavaScript Avanzado y Aplicaciones Web

Duración: 15 semanas  
Horas semanales: 30  
Total aproximado: 450 horas

---

# Objetivo del semestre

Profundizar en la programación con JavaScript y en la construcción
de aplicaciones web dinámicas capaces de interactuar con datos reales.

Durante este semestre el estudiante desarrollará habilidades en:

- programación avanzada con JavaScript
- manipulación dinámica del DOM
- manejo de datos en aplicaciones web
- programación asincrónica
- consumo de APIs externas
- almacenamiento de información en el navegador

El objetivo es transformar los conocimientos básicos adquiridos
en el semestre anterior en habilidades prácticas para el desarrollo
de aplicaciones web modernas.

Al finalizar el semestre el estudiante será capaz de:

- escribir código JavaScript avanzado
- trabajar con datos dinámicos
- consumir servicios externos mediante APIs
- manejar asincronía en aplicaciones web
- construir aplicaciones interactivas más complejas

---

# Asignaturas del semestre

## Programación Avanzada con JavaScript

Profundización en el lenguaje JavaScript y en la escritura
de código más estructurado y mantenible.

Temas principales:

- funciones avanzadas
- scope y closures
- programación modular
- arrays y objetos avanzados
- manejo de errores

Objetivo:

Desarrollar una comprensión más profunda del lenguaje
y mejorar la capacidad para resolver problemas complejos.

---

## Manipulación Avanzada del DOM

Control avanzado de interfaces web mediante JavaScript.

Temas principales:

- manipulación dinámica de elementos
- actualización de contenido en tiempo real
- eventos avanzados
- generación dinámica de interfaces

Objetivo:

Construir aplicaciones web altamente interactivas.

---

## Programación Asincrónica

Introducción al manejo de procesos que ocurren
de forma no inmediata en aplicaciones web.

Temas principales:

- callbacks
- promesas
- async / await
- control de operaciones asincrónicas

Objetivo:

Comprender cómo manejar procesos externos
como solicitudes a servidores o lectura de datos.

---

## Consumo de APIs

Conexión de aplicaciones web con servicios externos.

Temas principales:

- uso de Fetch API
- lectura de datos desde APIs públicas
- procesamiento de información externa
- integración de servicios web

Objetivo:

Aprender a trabajar con datos provenientes
de sistemas externos y servicios web.

---

## Manejo de Datos y Almacenamiento

Gestión de información dentro de aplicaciones web.

Temas principales:

- procesamiento de datos
- manipulación de estructuras de información
- almacenamiento en LocalStorage
- persistencia básica de datos

Objetivo:

Desarrollar aplicaciones capaces de guardar
y reutilizar información del usuario.

---

## Organización de Aplicaciones Web

Buenas prácticas para estructurar proyectos
de desarrollo de software.

Temas principales:

- organización de archivos
- modularización del código
- separación de responsabilidades
- escalabilidad básica

Objetivo:

Desarrollar proyectos más ordenados,
mantenibles y preparados para crecer.

---

# Proyecto del semestre

Construcción de una aplicación web dinámica.

Proyecto:

Dashboard de productividad personal.

Funcionalidades posibles:

- gestión de tareas
- consumo de APIs externas
- almacenamiento de datos del usuario
- visualización dinámica de información

Objetivos del proyecto:

- aplicar JavaScript avanzado
- trabajar con datos dinámicos
- integrar APIs externas
- construir una aplicación interactiva real
- ampliar el portafolio del estudiante

---

# Resultados esperados al finalizar el semestre

Al finalizar el semestre el estudiante tendrá:

- dominio intermedio de JavaScript
- comprensión de programación asincrónica
- capacidad de consumir APIs externas
- manejo de almacenamiento local
- experiencia construyendo aplicaciones dinámicas
- proyectos más complejos en su portafolio

# Semestre 2 — JavaScript Avanzado y Aplicaciones Web

Este semestre profundiza en la programación con JavaScript
y en la construcción de aplicaciones web dinámicas.

Durante estas **16 semanas** se desarrollarán habilidades
para trabajar con datos, asincronía y servicios externos.

Tecnologías utilizadas:

- JavaScript avanzado
- DOM avanzado
- Fetch API
- Promesas
- Async / Await
- APIs externas
- LocalStorage

El objetivo del semestre es pasar de escribir
scripts simples a construir aplicaciones web
dinámicas capaces de interactuar con datos reales.

---

# Contenido del semestre

Semana 1 — Repaso profundo de JavaScript  
Semana 2 — Funciones avanzadas  
Semana 3 — Scope y closures  
Semana 4 — Programación modular  
Semana 5 — Arrays avanzados  
Semana 6 — Objetos avanzados  
Semana 7 — Manejo de errores  

Semana 8 — Introducción a la asincronía  
Semana 9 — Promesas en JavaScript  
Semana 10 — Async / Await  
Semana 11 — Consumo de APIs  

Semana 12 — Manejo de datos en aplicaciones  
Semana 13 — Almacenamiento local (LocalStorage)

Semana 14 — Proyecto interactivo avanzado  
Semana 15 — Desarrollo del proyecto final  
Semana 16 — Finalización y presentación del proyecto

# Proyecto final del semestre

Aplicación:

Dashboard de productividad personal.

Funcionalidades posibles:

- gestión de tareas

- almacenamiento de datos

El proyecto integrará:

- JavaScript avanzado
- asincronía
- manipulación del DOM
- manejo de datos
- almacenamiento local

# Resultado del semestre

Al finalizar el semestre el estudiante podrá:

- escribir JavaScript avanzado
- trabajar con datos dinámicos
- consumir APIs externas
- manejar asincronía
- almacenar información en el navegador
- construir aplicaciones web interactivas

Esto prepara al estudiante para comenzar
el desarrollo **backend y aplicaciones completas**

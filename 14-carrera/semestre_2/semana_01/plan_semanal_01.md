# Semana 1 — Repaso Profundo de JavaScript

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Reforzar los conceptos fundamentales de JavaScript
aprendidos durante el semestre anterior.

Esta semana tiene como objetivo consolidar
las bases del lenguaje antes de avanzar
hacia conceptos más avanzados.

Durante la semana se revisarán:

- variables
- tipos de datos
- condicionales
- funciones
- arrays
- objetos

Esto permitirá comenzar el semestre
con una comprensión sólida del lenguaje.

---

# Habilidades que se desarrollarán

Al finalizar la semana el estudiante debería poder:

- escribir funciones con mayor claridad
- manipular arrays y objetos
- resolver problemas simples con JavaScript
- estructurar mejor el código

# Semana 11 — Consumo de APIs

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a conectar aplicaciones web con servicios externos usando **fetch** y **async / await**, integrando datos en tiempo real y mejorando la interactividad de la aplicación.

Durante la semana se estudiará:

- qué es una API y cómo funciona  
- consumo de APIs REST  
- procesamiento de respuestas JSON  
- manejo de errores en llamadas externas  
- integración de datos en interfaces dinámicas  

---

# Habilidades que se desarrollarán

Al finalizar la semana el estudiante debería poder:

- consumir datos de APIs externas usando fetch  
- integrar la información en la UI de manera dinámica  
- manejar errores y estados de carga  
- combinar datos de múltiples fuentes externas  
- documentar el flujo de consumo de APIs

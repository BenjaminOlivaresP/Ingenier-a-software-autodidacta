# Semana 5 — Arrays Avanzados

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Profundizar en el manejo de arrays y estructuras de datos en JavaScript.  
Esta semana permite manipular colecciones de datos de manera eficiente y preparar la base para objetos y estructuras más complejas.

Durante la semana se estudiará:

- métodos avanzados de arrays
- iteración y filtrado
- map, filter, reduce
- búsqueda y ordenamiento de datos
- manejo de arrays multidimensionales

Esto permitirá construir aplicaciones con procesamiento de datos más potente.

---

# Habilidades que se desarrollarán

Al finalizar la semana el estudiante debería poder:

- utilizar métodos avanzados de arrays
- manipular arrays complejos y multidimensionales
- combinar arrays y transformarlos
- resolver problemas usando técnicas funcionales

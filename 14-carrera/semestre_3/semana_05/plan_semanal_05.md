# Semana 5 — Bases de Datos Relacionales: conceptos

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Introducir los conceptos fundamentales de bases de datos relacionales y su importancia en sistemas backend.  
El estudiante comprenderá cómo organizar, almacenar y relacionar información de manera eficiente para construir aplicaciones robustas.

Durante la semana se revisarán:

- conceptos de bases de datos relacionales
- tablas, filas y columnas
- claves primarias y foráneas
- normalización de datos
- relaciones entre tablas (1:1, 1:N, N:N)

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- comprender la estructura de una base de datos relacional  
- diseñar tablas con claves primarias y foráneas  
- establecer relaciones entre tablas  
- aplicar principios básicos de normalización  
- planificar la base de datos para proyectos backend

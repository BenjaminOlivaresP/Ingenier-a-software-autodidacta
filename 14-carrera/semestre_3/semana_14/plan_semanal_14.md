# Semana 14 — Proyecto intermedio: API funcional

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Construir un proyecto intermedio que integre todos los conocimientos de backend hasta la fecha.  
El estudiante desarrollará una API funcional con Node.js, Express y SQL, incluyendo endpoints completos, manejo de errores y documentación básica.

Durante la semana se trabajará en:

- diseño de endpoints RESTful  
- conexión de la API con la base de datos  
- implementación de operaciones CRUD  
- pruebas de la API con Postman  
- documentación básica de la API  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- diseñar y estructurar una API funcional  
- crear endpoints CRUD con Node.js y SQL  
- manejar errores de manera adecuada  
- documentar la API para uso propio y de otros desarrolladores  
- probar la API y asegurar su correcto funcionamiento

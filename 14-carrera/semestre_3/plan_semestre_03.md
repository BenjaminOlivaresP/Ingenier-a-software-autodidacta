# Semestre 3 — Backend y Arquitectura Web

Duración: 16 semanas  
Horas semanales: 30  
Total aproximado: 480 horas

---

# Objetivo del semestre

Aprender a construir sistemas backend completos
y comprender la arquitectura de aplicaciones web modernas.

Durante este semestre el estudiante desarrollará habilidades en:

- desarrollo de servidores y APIs con Node.js  
- manejo de bases de datos relacionales (SQL)  
- construcción de endpoints RESTful  
- autenticación y seguridad básica  
- pruebas y despliegue de aplicaciones  
- buenas prácticas de organización de proyectos

El objetivo es transformar el conocimiento de frontend
y aplicaciones dinámicas en habilidades completas de backend.

Al finalizar el semestre el estudiante será capaz de:

- crear servidores y rutas con Node.js  
- conectar aplicaciones a bases de datos SQL  
- diseñar APIs RESTful funcionales  
- implementar autenticación y manejo básico de seguridad  
- probar y desplegar aplicaciones backend  
- integrar backend con frontend dinámico

---

# Asignaturas del semestre

## Programación Backend con Node.js

Profundización en la programación de servidores
y en la construcción de aplicaciones backend.

Temas principales:

- Node.js básico y avanzado  
- creación de servidores HTTP  
- rutas y manejo de solicitudes  
- middleware y control de flujo  
- POO en backend

Objetivo:

Desarrollar habilidades para construir aplicaciones
server-side estructuradas y mantenibles.

---

## Bases de Datos Relacionales y SQL

Gestión de información mediante bases de datos relacionales.

Temas principales:

- conceptos de bases de datos  
- diseño de tablas y relaciones  
- consultas SQL: SELECT, INSERT, UPDATE, DELETE  
- conexión de Node.js con bases de datos  
- integridad de datos

Objetivo:

Aprender a almacenar, consultar y manipular información
de manera eficiente y segura.

---

## APIs RESTful y Consumo de Servicios

Construcción de servicios web y endpoints para aplicaciones.

Temas principales:

- diseño de APIs RESTful  
- métodos HTTP: GET, POST, PUT, DELETE  
- integración de APIs con frontend  
- manejo de errores y status codes  
- documentación de APIs

Objetivo:

Construir APIs funcionales y seguras
para aplicaciones web modernas.

---

## Seguridad y Autenticación

Introducción a la protección de aplicaciones
y gestión de usuarios.

Temas principales:

- autenticación básica  
- control de acceso  
- manejo de contraseñas  
- middleware de seguridad  
- protección de endpoints

Objetivo:

Desarrollar aplicaciones más seguras y controladas
para uso real.

---

## Testing y Despliegue

Pruebas y entrega de aplicaciones backend.

Temas principales:

- testing básico de rutas y funciones  
- manejo de errores y logging  
- despliegue local y en entornos de prueba  
- buenas prácticas de desarrollo

Objetivo:

Garantizar la calidad y estabilidad de las aplicaciones
antes de su integración con frontend.

---

## Organización de Proyectos Backend

Buenas prácticas para estructurar proyectos de software.

Temas principales:

- organización de archivos  
- modularización del código  
- separación de responsabilidades  
- escalabilidad básica

Objetivo:

Mantener proyectos ordenados y mantenibles,
preparados para crecer y trabajar en equipo.

---

# Proyecto del semestre

Construcción de un sistema backend completo.

Proyecto:

Sistema de gestión de usuarios y datos con API.

Funcionalidades posibles:

- endpoints para CRUD completo  
- autenticación básica de usuarios  
- almacenamiento seguro de información  
- integración con frontend dinámico

Objetivos del proyecto:

- aplicar Node.js y Express  
- conectar y manejar bases de datos SQL  
- diseñar APIs funcionales  
- implementar buenas prácticas de backend  
- construir un proyecto real para el portafolio

El desarrollo del proyecto se realizará
durante las últimas semanas del semestre
para integrar todos los conocimientos aprendidos.

---

# Resultados esperados al finalizar el semestre

Al finalizar el semestre el estudiante tendrá:

- dominio de Node.js y programación backend  
- comprensión de bases de datos relacionales y SQL  
- capacidad para diseñar y consumir APIs RESTful  
- experiencia en autenticación y seguridad básica  
- conocimiento de testing y despliegue de aplicaciones  
- proyectos backend completos listos para integrarse con frontend

# Semana 13 — Despliegue de aplicaciones locales

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a preparar y ejecutar aplicaciones backend en entornos locales.  
El estudiante desarrollará habilidades para configurar servidores, administrar bases de datos locales y garantizar que su aplicación funcione correctamente antes de un despliegue real.

Durante la semana se trabajará en:

- configuración de entornos de desarrollo  
- despliegue de aplicaciones Node.js en local  
- manejo de bases de datos locales  
- variables de entorno y configuración  
- pruebas funcionales de la aplicación  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- configurar un servidor local para Node.js  
- administrar bases de datos SQL locales  
- manejar variables de entorno para seguridad y flexibilidad  
- realizar pruebas funcionales completas  
- preparar la aplicación para despliegues posteriores  

---

Bloque 1 — 11:00 a 13:00  
Teoría y configuración de entorno local.

# Día 1 — Configuración de entorno local

## Bloque 1 — Conceptos

Temas:

- instalación de Node.js y gestor de paquetes  
- instalación de bases de datos locales (MySQL / PostgreSQL)  
- configuración de editores y terminales  

Actividad:

Documento:  
"Preparación del entorno local de desarrollo"

---

## Bloque 2 — Ejercicios

Práctica:

- crear proyecto Node.js básico  
- configurar base de datos local de ejemplo  
- probar conexión desde la aplicación  

---

# Día 2 — Variables de entorno y configuración

## Bloque 1 — Conceptos

Temas:

- uso de `.env` y `dotenv`  
- variables de entorno para seguridad y flexibilidad  
- configuración de rutas y credenciales  

Actividad:

Documento:  
"Variables de entorno en aplicaciones Node.js"

---

## Bloque 2 — Ejercicios

Práctica:

- crear archivo `.env` con credenciales de base de datos  
- configurar conexión de Node.js usando variables de entorno  
- probar acceso seguro a la base de datos  

---

# Día 3 — Despliegue de aplicación local

## Bloque 1 — Conceptos

Temas:

- ejecución de la aplicación en local

Actividad:

Documento:  
"Despliegue local de aplicaciones Node.js"

---

## Bloque 2 — Ejercicios

Práctica:

---

## Bloque 1 — Conceptos

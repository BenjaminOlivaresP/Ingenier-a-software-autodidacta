# Semana 2 — Servidores HTTP y Rutas

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a crear servidores HTTP con Node.js y gestionar rutas básicas para servir contenido o responder solicitudes.  
Esta semana se enfoca en establecer la comunicación entre cliente y servidor mediante rutas, métodos HTTP y manejo de solicitudes simples.

Durante la semana se revisarán:

- creación de servidores con Node.js
- rutas estáticas y dinámicas
- métodos HTTP: GET, POST, PUT, DELETE
- manejo de parámetros y query strings
- herramientas para pruebas de APIs

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- levantar servidores HTTP en Node.js  
- definir rutas para diferentes endpoints  
- manejar parámetros en rutas  
- responder correctamente a solicitudes GET y POST  
- probar rutas con herramientas como Postman

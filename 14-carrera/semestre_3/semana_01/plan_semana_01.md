# Semana 1 — Introducción a Backend y Arquitectura Web

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Familiarizar al estudiante con los conceptos fundamentales del desarrollo backend y la arquitectura de aplicaciones web.  
Esta semana sirve como puente entre el frontend avanzado del Semestre 2 y la construcción de sistemas completos en el backend.

Durante la semana se revisarán:

- conceptos de servidor y cliente
- HTTP y protocolos web
- rutas y endpoints
- introducción a Node.js
- estructura básica de aplicaciones backend

Esto permitirá al estudiante iniciar el semestre con una comprensión sólida de cómo funcionan las aplicaciones web completas.

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- comprender la comunicación cliente-servidor
- identificar componentes de un backend
- iniciar proyectos con Node.js
- entender la estructura de rutas y endpoints
- relacionar el frontend con servicios backend

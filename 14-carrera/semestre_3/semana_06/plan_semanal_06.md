# Semana 6 — SQL básico: consultas y operaciones

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a utilizar SQL para consultar y manipular bases de datos relacionales.  
El estudiante adquirirá habilidades para realizar operaciones básicas de lectura, inserción, actualización y eliminación de datos, fundamentales para cualquier sistema backend.

Durante la semana se revisarán:

- sentencias SELECT, INSERT, UPDATE y DELETE  
- filtros y condiciones con WHERE  
- ordenamiento de resultados  
- funciones agregadas (COUNT, SUM, AVG)  
- JOINs simples para relacionar tablas  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- escribir consultas SQL básicas y eficientes  
- filtrar y ordenar datos  
- combinar información de varias tablas  
- actualizar y eliminar registros de manera segura  
- aplicar funciones agregadas para análisis de datos

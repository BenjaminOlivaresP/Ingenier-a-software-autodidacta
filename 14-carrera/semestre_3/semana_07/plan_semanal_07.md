# Semana 7 — Conexión de Node.js con bases de datos

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a integrar Node.js con bases de datos relacionales.  
El estudiante desarrollará la habilidad de conectar un servidor backend a una base de datos SQL, ejecutar consultas desde el código y manejar resultados de manera eficiente y segura.

Durante la semana se revisarán:

- instalación y configuración de drivers de base de datos en Node.js  
- conexión a bases de datos SQL desde Node.js  
- ejecución de consultas SELECT, INSERT, UPDATE y DELETE  
- manejo de errores en operaciones de base de datos  
- uso de parámetros y prepared statements para seguridad  

---

# Habilidades que se desarrollarán

Al finalizar la semana, el estudiante podrá:

- configurar una base de datos para un proyecto Node.js  
- conectar Node.js con bases de datos relacionales  
- ejecutar consultas y procesar resultados en JavaScript  
- manejar errores y excepciones en operaciones de datos  
- aplicar buenas prácticas de seguridad y validación

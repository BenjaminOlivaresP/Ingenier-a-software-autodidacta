# Semestre 3 — Backend y Arquitectura Web

Este semestre se centra en la construcción de sistemas backend
y en comprender la arquitectura de aplicaciones web modernas.

Durante estas **16 semanas** se desarrollarán habilidades
para manejar datos en el servidor, construir APIs y
organizar proyectos de software de manera profesional.

Tecnologías utilizadas:

- Node.js  
- Express  
- SQL y bases de datos relacionales  
- RESTful APIs  
- Control de versiones avanzado (Git / GitHub)  
- Arquitectura básica de aplicaciones web  

El objetivo del semestre es pasar de construir aplicaciones
frontend dinámicas a crear sistemas completos que gestionen
datos, servicios y funcionalidades en el servidor.

---

# Contenido del semestre

Semana 1 — Introducción a backend y Node.js  
Semana 2 — Servidores HTTP y rutas  
Semana 3 — Middleware y manejo de solicitudes  
Semana 4 — Programación orientada a objetos en backend  
Semana 5 — Bases de datos relacionales: conceptos  
Semana 6 — SQL básico: consultas y operaciones  
Semana 7 — Conexión de Node.js con bases de datos  

Semana 8 — Autenticación y seguridad básica  
Semana 9 — APIs RESTful: diseño y consumo  
Semana 10 — CRUD completo con Node.js y SQL  
Semana 11 — Manejo de errores y logging  

Semana 12 — Testing básico de aplicaciones  
Semana 13 — Despliegue de aplicaciones locales  
Semana 14 — Proyecto intermedio: API funcional  
Semana 15 — Desarrollo del proyecto final  
Semana 16 — Finalización y presentación del proyecto

---

# Proyecto final del semestre

Aplicación:

Sistema backend completo con API.

Funcionalidades posibles:

- gestión de usuarios y datos  
- autenticación básica  
- endpoints para consumir datos  
- operaciones CRUD completas  
- integración con bases de datos  

El proyecto integrará:

- Node.js y Express  
- SQL y almacenamiento de datos  
- diseño de APIs RESTful  
- manejo de errores y buenas prácticas de backend  

---

# Resultado del semestre

Al finalizar el semestre el estudiante podrá:

- crear servidores y rutas con Node.js  
- conectar aplicaciones a bases de datos SQL  
- diseñar y consumir APIs RESTful  
- implementar autenticación básica  
- manejar errores y registros  
- construir sistemas backend completos listos para integrarse con frontend  

Esto prepara al estudiante para continuar con **desarrollo full stack
y aplicaciones completas** en los semestres siguientes.

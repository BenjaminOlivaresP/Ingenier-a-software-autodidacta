# Semana 1 — Fundamentos de la Ingeniería de Software

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Comprender los fundamentos de la computación y comenzar a
desarrollar pensamiento computacional mientras se construye
la primera página web estructurada.

Esta semana establece las bases mentales del programador:

- comprender qué es realmente un computador
- entender qué es el software
- aprender a descomponer problemas
- comenzar a construir interfaces web
- aprender a documentar el aprendizaje

---

# Habilidades que se desarrollarán

Al finalizar la semana deberías poder:

- explicar qué es hardware y software
- comprender qué es un algoritmo
- explicar cómo funciona internet a alto nivel
- estructurar una página web usando HTML
- aplicar estilos básicos con CSS
- documentar tu proceso de aprendizaje

---

# Estructura diaria

Bloque 1 — 11:00 a 13:00  
Fundamentos teóricos y pensamiento computacional.

Bloque 2 — 16:00 a 19:00  
Programación práctica y construcción de proyectos.

# Día 1 — Introducción a la computación

## Bloque 1 — Fundamentos

Temas:

- qué es un computador
- hardware vs software
- qué es un programa
- qué es un algoritmo
- cómo ejecuta instrucciones un computador

Actividad:

Escribir un documento:

"¿Qué hace realmente un computador?"

Extensión recomendada: 200–400 palabras.

---

## Bloque 2 — Primer proyecto web

Crear el primer proyecto del semestre.

Proyecto:

"Mi primera página profesional"

Estructura mínima:

index.html

Debe contener:

- título
- encabezado
- sección principal
- footer

Objetivo:

Comprender la estructura básica de una página web.

---

# Día 2 — Historia de la programación

## Bloque 1 — Historia de la computación

Temas:

- origen de los computadores
- primeras máquinas programables
- primeros lenguajes de programación
- nacimiento de la ingeniería de software

Actividad:

Escribir un documento:

"Por qué existe la programación".

---

## Bloque 2 — Expansión del proyecto web

Mejorar la página creada.

Agregar:

- múltiples secciones
- estructura semántica
- imágenes
- textos de contenido

Objetivo:

Comprender cómo crece una página web.

---

# Día 3 — Pensamiento computacional

## Bloque 1 — Pensar como programador

Temas:

- descomposición de problemas
- abstracción
- reconocimiento de patrones
- diseño de algoritmos

Ejercicio:

Resolver 3 problemas simples usando pseudocódigo.

Ejemplos:

- calcular promedio de números
- decidir si un número es par o impar
- ordenar una lista simple

---

## Bloque 2 — Introducción a CSS

Aprender fundamentos de estilo visual.

Temas:

- selectores
- clases
- colores
- tipografía
- tamaños

Actividad:

Aplicar estilos básicos a la página creada.

Objetivo:

Separar **estructura (HTML)** de **presentación (CSS)**.

---

## Bloque 2 — Mejora visual del proyecto

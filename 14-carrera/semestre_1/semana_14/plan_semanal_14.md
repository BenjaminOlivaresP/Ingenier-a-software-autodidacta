# Semana 14 — Proyecto interactivo

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Construir una aplicación web interactiva
utilizando todo lo aprendido durante el semestre.

Tecnologías utilizadas:

- HTML
- CSS
- JavaScript
- DOM
- Eventos

El objetivo es consolidar conocimientos creando
un sistema funcional que responda a acciones del usuario.

---

# Habilidades que se desarrollarán

Al finalizar la semana deberías poder:

- planificar una pequeña aplicación
- estructurar HTML correctamente
- aplicar estilos con CSS
- usar JavaScript para manipular la página
- manejar eventos del usuario

---

# Estructura diaria

Bloque 1 — 11:00 a 13:00  
Diseño y planificación del proyecto.

Bloque 2 — 16:00 a 19:00  
Construcción del sistema.

# Proyecto de la semana

Aplicación:

**Panel de objetivos personales**

El usuario podrá:

- escribir objetivos
- agregarlos a una lista
- eliminarlos
- marcarlos como completados

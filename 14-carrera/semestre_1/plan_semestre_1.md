# Semestre 1 — Fundamentos de la Ingeniería de Software

Duración: 15 semanas  
Horas semanales: 30  
Total aproximado: 450 horas

---

# Objetivo del semestre

Construir las bases fundamentales necesarias para iniciar
una carrera sólida en ingeniería de software.

Durante este semestre se desarrollarán habilidades en:

- pensamiento computacional
- lógica aplicada a programación
- comprensión del funcionamiento del software
- fundamentos del desarrollo web
- uso de herramientas profesionales de programación

Al finalizar el semestre el estudiante será capaz de:

- comprender cómo funciona un sistema de software
- pensar en términos de algoritmos
- construir páginas web estructuradas
- escribir código básico en JavaScript
- desarrollar su primera aplicación web funcional

---

# Asignaturas del semestre

## Historia y Fundamentos de la Computación

Introducción al origen y evolución de la computación.

Temas principales:

- origen de los computadores
- primeros lenguajes de programación
- evolución del software
- nacimiento de internet
- impacto de la programación en la sociedad

Objetivo:

Desarrollar comprensión del contexto histórico
y del campo profesional de la ingeniería de software.

---

## Pensamiento Computacional

Desarrollo de la forma de pensar utilizada por los programadores.

Temas principales:

- algoritmos
- descomposición de problemas
- patrones
- abstracción
- diseño de soluciones

Objetivo:

Aprender a transformar problemas complejos
en soluciones programables.

---

## Lógica Matemática para Programación

Introducción a los principios lógicos utilizados en programación.

Temas principales:

- lógica proposicional
- operadores lógicos
- condiciones
- razonamiento estructurado

Objetivo:

Desarrollar bases formales para el pensamiento algorítmico.

---

## Fundamentos de Programación

Primer contacto con la programación real.

Tecnología principal:

JavaScript

Temas principales:

- variables
- condicionales
- funciones
- estructuras básicas
- resolución de problemas simples

---

## Introducción al Desarrollo Web

Fundamentos del desarrollo de interfaces web.

Tecnologías:

HTML  
CSS  

Temas principales:

- estructura de páginas web
- etiquetas semánticas
- diseño y estilos
- layouts básicos

---

## Herramientas del Programador

Uso de herramientas profesionales utilizadas en desarrollo de software.

Herramientas:

- Visual Studio Code
- Git
- GitHub
- organización de proyectos

Objetivo:

Aprender a trabajar con herramientas reales de ingeniería de software.

---

# Proyecto del semestre

Construcción de una aplicación web real.

Proyecto:

Aplicación de seguimiento de hábitos.

Objetivos del proyecto:

- aplicar conocimientos de HTML, CSS y JavaScript
- aprender organización de código
- desarrollar una herramienta funcional
- construir el primer proyecto de portafolio

---

# Resultados esperados al finalizar el semestre

Al finalizar el semestre el estudiante tendrá:

- bases sólidas en pensamiento computacional
- conocimientos fundamentales de programación
- habilidades iniciales en desarrollo web
- experiencia usando herramientas profesionales
- un proyecto funcional para su portafolio

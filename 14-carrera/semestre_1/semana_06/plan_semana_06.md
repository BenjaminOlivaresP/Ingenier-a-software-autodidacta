Duración: 5 días

# Objetivo de la semana

Aprender a utilizar estructuras condicionales en JavaScript
para que los programas puedan tomar decisiones según
las condiciones que se cumplan.

Durante esta semana se aprenderá:

- qué es una condición
- operadores de comparación
- estructura if
- estructura if / else
- estructura if / else if

El objetivo es que el programa pueda reaccionar
de forma diferente según la información recibida.

# Día 1 — Comparaciones

## Bloque 1 — Operadores de comparación

Temas:

- mayor que >
- menor que <
- mayor o igual >=
- menor o igual <=
- igual ==
- diferente !=

Ejemplos:

edad > 17

edad < 30

# Día 3 — if / else

## Bloque 1 — Dos caminos posibles

- if
- else

}

# Día 4 — else if

## Bloque 1 — Múltiples condiciones

Temas:

- múltiples decisiones
- encadenamiento de condiciones

Ejemplo:

let nota = 85

if (nota >= 90) {
console.log("Excelente")
} else if (nota >= 70) {
console.log("Aprobado")
} else {
console.log("Reprobado")
}

Duración: 5 días  
Horas diarias: 6

# Objetivo de la semana

Aprender a utilizar bucles (loops) en JavaScript
para repetir instrucciones automáticamente.

Los bucles permiten ejecutar una misma acción
muchas veces sin tener que escribir el código repetidamente.

Durante esta semana se aprenderá:

- qué es un bucle
- bucle for
- bucle while
- control de iteraciones
- repetición de tareas

# Día 1 — Qué es un bucle

## Bloque 1 — Introducción a la repetición

Temas:

- qué es un bucle
- repetición en programación
- automatización de tareas

Ejemplo sin bucle:

console.log("Hola")
console.log("Hola")
console.log("Hola")

Ejemplo con bucle:

for (let i = 0; i < 3; i++) {
console.log("Hola")
}

Actividad:

Documento:

"Qué es un bucle y para qué sirve".

---

## Bloque 2 — Primer loop

Crear bucle que imprima números del 1 al 10.

Ejemplo:

for (let i = 1; i <= 10; i++) {
console.log(i)
}

Objetivo:

Comprender cómo funciona una repetición automática.

---

# Día 2 — Estructura del bucle for

## Bloque 1 — Anatomía del loop

Temas:

- variable de control
- condición
- incremento

Estructura:

for (inicio; condición; incremento)

Ejemplo:

for (let i = 0; i < 5; i++) {
console.log(i)
}

Actividad:

Documento:

"Cómo funciona el bucle for".

# Día 3 — Bucle while

## Bloque 1 — Introducción a while

Temas:

- bucle basado en condición
- diferencia entre for y while

Ejemplo:

while (i <= 5) {
console.log(i)
i++
}

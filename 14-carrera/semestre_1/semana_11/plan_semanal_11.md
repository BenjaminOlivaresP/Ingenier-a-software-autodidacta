# Semana 11 — Arrays de Objetos en JavaScript

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a combinar arrays y objetos para manejar
estructuras de datos más complejas.

Esta combinación permite almacenar múltiples entidades,
cada una con sus propias propiedades.

Durante esta semana se aprenderá:

- arrays de objetos
- recorrer arrays con objetos
- acceder a propiedades dentro de listas
- modificar elementos dentro de arrays

El objetivo es manejar estructuras de datos similares
a las utilizadas en aplicaciones reales.

- crear arrays de objetos

# Semana 05 — Introducción a JavaScript

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Comenzar a programar utilizando JavaScript,
uno de los lenguajes más importantes del desarrollo web.

Durante esta semana se aprenderá:

- qué es JavaScript
- cómo ejecutar código en el navegador
- cómo declarar variables
- cómo manipular información básica

El objetivo es escribir los primeros programas funcionales
y comprender cómo los sitios web pueden reaccionar a acciones del usuario.

---

# Habilidades que se desarrollarán

Al finalizar la semana deberías poder:

- comprender qué es JavaScript
- ejecutar código en el navegador
- declarar variables
- realizar operaciones básicas
- escribir scripts simples

---

Bloque 2 — 16:00 a 19:00

# Día 1 — Qué es JavaScript

## Bloque 1 — Introducción al lenguaje

Temas:

- qué es JavaScript
- diferencia entre HTML, CSS y JavaScript
- dónde se ejecuta JavaScript
- cómo funciona dentro del navegador

Actividad:

Escribir documento:

"Qué es JavaScript y por qué es importante en el desarrollo web".

Extensión recomendada: 300–500 palabras.

---

## Bloque 2 — Primer script

Crear archivo:

script.js

Conectar el archivo al HTML del sitio.

Ejemplo dentro del HTML:

<script src="script.js"></script>

Primer código:

console.log("Hola mundo desde JavaScript");

Objetivo:

Ver el mensaje dentro de la consola del navegador.

---

# Día 2 — Variables

## Bloque 1 — Declaración de variables

Temas:

- qué es una variable
- almacenar información
- tipos de datos básicos

Tipos iniciales:

- string (texto)
- number (números)
- boolean (verdadero o falso)

Ejemplo:

let nombre = "Benjamín"
let edad = 25
let entrenando = true

Actividad:

Documento:

"Cómo funcionan las variables en JavaScript".

---

## Bloque 2 — Práctica con variables

Crear variables como:

- nombre
- edad
- ciudad
- objetivo

Imprimirlas en consola.

Ejemplo:

console.log(nombre)
console.log(edad)

Objetivo:

Comprender cómo se guarda información.

---

# Día 3 — Operaciones

## Bloque 1 — Operaciones matemáticas

Temas:

- suma
- resta
- multiplicación
- división

Ejemplo:

let a = 10
let b = 5

console.log(a + b)
console.log(a - b)
console.log(a * b)
console.log(a / b)

Actividad:

Documento:

"Operaciones matemáticas en JavaScript".

---

## Bloque 2 — Calculadora simple

Crear un pequeño script que:

- declare dos números
- calcule suma
- calcule resta
- calcule multiplicación
- calcule división

Mostrar resultados en consola.

---

# Día 4 — Interacción básica

## Bloque 1 — Entrada de información

Temas:

- interacción con el usuario
- función prompt()

Ejemplo:

let nombre = prompt("¿Cómo te llamas?")
console.log("Hola " + nombre)

Actividad:

Documento:

"Cómo los programas reciben información del usuario".

---

## Bloque 2 — Script interactivo

Crear script que:

- pregunte nombre
- pregunte edad
- muestre mensaje personalizado

Ejemplo:

Hola Benjamín, tienes 25 años.

Objetivo:

Crear interacción básica con el usuario.

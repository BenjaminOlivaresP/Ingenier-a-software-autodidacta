# Semana 15 — Proyecto Final del Semestre 1

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Desarrollar un proyecto final que integre todos los conocimientos
aprendidos durante el primer semestre de la carrera.

Tecnologías utilizadas:

- HTML
- CSS
- JavaScript
- DOM
- Eventos

El objetivo es demostrar que se puede construir
una aplicación web simple pero completamente funcional.

---

# Habilidades que se consolidarán

Al finalizar la semana deberías poder:

- estructurar una página web completa
- aplicar estilos correctamente
- programar lógica con JavaScript
- manipular el DOM
- manejar interacción del usuario

---

# Proyecto final

Aplicación:

**Gestor de objetivos personales**

La aplicación permitirá:

- agregar objetivos
- marcar objetivos como completados
- eliminar objetivos
- visualizar lista de metas

Este proyecto demuestra la integración de todos los conocimientos.

# Semana 09 — Arrays en JavaScript

Duración: 5 días  
Horas diarias: 6  
Total semanal: 30 horas

---

# Objetivo de la semana

Aprender a trabajar con arrays en JavaScript para
almacenar y manipular múltiples datos dentro
de una misma estructura.

Los arrays permiten manejar listas de información
de manera organizada y eficiente.

Durante esta semana se aprenderá:

- qué es un array
- cómo crear arrays
- cómo acceder a sus elementos
- recorrer arrays con loops
- modificar arrays

El objetivo es comenzar a manejar conjuntos
de datos como lo hacen los programas reales.

---

# Habilidades que se desarrollarán

Al finalizar la semana deberías poder:

- crear arrays
- acceder a elementos dentro de un array
- recorrer arrays con loops
- agregar y eliminar elementos
- utilizar arrays para resolver problemas

## Bloque 1 — Loops con arrays

Temas:

- recorrer listas
- usar loops con arrays

Ejemplo:

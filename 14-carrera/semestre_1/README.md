# Semestre 1 — Fundamentos del Desarrollo Web

Este semestre introduce los principios fundamentales del desarrollo web.

Durante estas 15 semanas se estudian las bases necesarias para comprender
cómo se construyen las páginas web modernas.

Tecnologías utilizadas:

- HTML
- CSS
- JavaScript
- DOM
- Eventos

El objetivo del semestre es pasar de comprender
cómo funcionan las páginas web a poder construir
aplicaciones web simples de manera autónoma.

---

# Contenido del semestre

Semana 1 — Introducción al desarrollo web  
Semana 2 — Estructura de HTML  
Semana 3 — Elementos y etiquetas HTML  
Semana 4 — Formularios HTML  
Semana 5 — Introducción a CSS  
Semana 6 — Diseño y estilos  
Semana 7 — Layout y estructura visual  

Semana 8 — Introducción a JavaScript  
Semana 9 — Variables y tipos de datos  
Semana 10 — Condicionales y lógica  
Semana 11 — Arrays y objetos  

Semana 12 — Manipulación del DOM  
Semana 13 — Eventos en JavaScript  
Semana 14 — Proyecto interactivo  
Semana 15 — Proyecto final

---

# Proyecto final del semestre

Aplicación:

Gestor de objetivos personales.

Funcionalidades:

- agregar objetivos
- marcar objetivos como completados
- eliminar objetivos
- interacción dinámica con JavaScript

Este proyecto integra todos los conocimientos
aprendidos durante el semestre.

---

# Resultado del semestre

Al finalizar el semestre el estudiante puede:

- construir páginas web completas
- aplicar estilos con CSS
- programar lógica con JavaScript
- manipular el DOM
- crear interacciones con el usuario

Esto establece la base para continuar con
JavaScript avanzado y desarrollo web moderno.

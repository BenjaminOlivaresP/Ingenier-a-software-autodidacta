# Diario de Ingeniería

Fecha: 13 Marzo 2026

---

## Inicio de la Ingeniería de Software Autodidacta

Hoy se inició oficialmente la construcción del sistema de estudio
para convertirme en ingeniero de software de forma autodidacta.

Durante esta sesión se logró:

- crear el repositorio del proyecto
- configurar Git
- conectar el repositorio con GitHub
- crear la arquitectura de la carrera
- definir la malla curricular
- estructurar el semestre 1
- crear el sistema de seguimiento de progreso

---

## Aprendizajes técnicos

Aprendí a utilizar Git para:

- inicializar repositorios
- hacer commits
- subir cambios a GitHub
- resolver conflictos iniciales de repositorio

Esto me permitió comprender cómo los programadores gestionan
proyectos de software en entornos profesionales.

---

## Reflexión

Este es el inicio de un proceso largo de aprendizaje
que tiene como objetivo dominar el desarrollo de software
y poder trabajar creando herramientas reales.

El objetivo es mantener disciplina diaria,
construir proyectos reales
y desarrollar mentalidad de ingeniero.

## Algoritmo y su funcion con Ejemplo simple:

problema:   Calcular el promedio de tres números.
        Un humano podría hacerlo mentalmente, pero un 
        programar un computador necesita pasos exactos:

-pedir el primer número
-pedir el segundo número
-pedir el tercer número
-sumar los tres números
-dividir el resultasdo entre 3
-mostrar el resultado 
  
(eso ya es un algoritmo.)

---

## Forma de escribir algoritmos (pseudocódigo)
Los programadores suelen escribir primero pseudocódigo.
Este es un lenguaje intermedio entre el español y programación.

ejemplo:

INICIO

Leer numero1
Leer numero2
Leer numero3

suma = numero1 + numero2 + numero3

promedio = suma / 3

mostrar promedio

FIN
(esto no es código pero ya es lógica programable.)

## Ejercicio 1 — Determinar si un número es par o impar

Problema:
Determinar si un número ingresado por el usuario es par o impar.

Pseudocódigo:

INICIO

leer numero

si numero % 2 == 0 entonces
    mostrar "El número es par"
si no
    mostrar "El número es impar"

## Nota 2

Estos ejercicios son de "Lógica condicional".

En programación se les conoce como:

"Estructuras de Control"

Las más importantes son:

-Secuencia (pasos normales)
-Condiciones (si pasa algo, hacer algo)
-Repeticiones (hacer algo muchas veces)

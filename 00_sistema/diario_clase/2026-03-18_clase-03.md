# Materia bloque 1

## Pensamiento computacional

El pensamiento computacional es la capacidad de resolver problemas de forma lógica, estructurada y paso a paso, de manera que un computador pueda ejecutar la solución.

No se trata de programar directamente, sino de aprender a pensar como programador.

---

## Pilares del pensamiento computacional

### 1. Descomposición

Consiste en dividir un problema grande en partes más pequeñas y manejables.

Ejemplo:

Calcular un promedio:

- obtener números  
- sumarlos  
- dividir el resultado  

Esto permite simplificar problemas complejos.

---

### 2. Abstracción

Consiste en enfocarse en los elementos importantes e ignorar los detalles innecesarios.

Ejemplo:

Para determinar si un número es par:

- no importa el valor exacto  
- solo importa si es divisible por 2  

---

### 3. Reconocimiento de patrones

Consiste en identificar similitudes entre distintos problemas.

Ejemplo:

- promedio de 3 números  
- promedio de 10 números  

Ambos siguen la misma lógica, solo cambia la cantidad de datos.

---

### 4. Diseño de algoritmos

Consiste en crear una secuencia ordenada de pasos para resolver un problema.

Un algoritmo debe ser:

- claro  
- finito  
- ordenado  
- ejecutable

## Introducción a CSS

En el segundo bloque se comenzó a trabajar con estilos visuales utilizando CSS.

CSS permite separar la estructura de una página web (HTML) de su presentación visual.

---

## Conceptos fundamentales

Se trabajaron los siguientes conceptos:

### Selectores

Permiten seleccionar elementos HTML para aplicar estilos.

Ejemplo:

- etiquetas  
- clases  
- identificadores  

---

### Clases

Permiten reutilizar estilos en múltiples elementos.

Esto facilita la organización del diseño.

---

### Propiedades básicas

Se aplicaron estilos como:

- colores  
- tipografía  
- tamaños  
- espaciado  

---

## Aplicación en el proyecto

Se aplicaron estilos al proyecto web desarrollado anteriormente.

Se mejoraron aspectos como:

- colores de fondo  
- alineación de contenido  
- jerarquía visual  
- separación de elementos

## Objetivo técnico

- HTML → estructura
